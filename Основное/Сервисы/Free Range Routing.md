> [!INFO] Что такое **Free Range Routing (FRR)**?
Программный пакет для сетевой маршрутизации, который обеспечивает реализацию различных протоколов динамической маршрутизации на Unix-подобных системах, таких как Linux.

Установка пакета при помощи команды:
```bash
apt-get update && apt-get install frr -y
```
## Настройка демонов FRR
Конфигурация активных демонов редактируется в файле:
```bash
vim /etc/frr/daemons
````

В этом файле можно включать или отключать нужные службы, например:

```bash
zebra=yes # включает(yes)/отключает(no) службу Zebra, которая управляет таблицами маршрутизации и взаимодействует с другими демонами FRR.
bgpd=yes  # включает(yes)/отключает(no) службу BGP Daemon, которая отвечает за маршрутизацию по протоколу BGP (Border Gateway Protocol).
ospfd=no # включает(yes)/отключает(no) службу OSPF Daemon (ospfd)**, которая используется для маршрутизации по OSPF
```

После внесения изменений нужно перезапустить службу:

```bash
systemctl restart frr
```

## Работа с FRR CLI

Для работы с FRR используется командная строка `vtysh`. Войти в неё можно так:

```bash
vtysh
```

После входа доступны команды конфигурирования, например:

```bash
configure terminal
router bgp 65001
 neighbor 192.168.1.1 remote-as 65002
```

## Проверка состояния

Просмотреть активные маршруты можно командами:

```bash
show ip route
show bgp summary
```
