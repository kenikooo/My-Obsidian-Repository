> [!INFO] Что такое **Cyber Backup (Кибер Бэкап)**?
> Комплексное программное решение от компании "Киберпротект" для резервного копирования, восстановления и защиты данных от вирусов-шифровальщиков для физических, виртуальных и облачных сред, включая российское ОС, СУБД и приложения. Это универсальная система, которая позволяет централизованно управлять защитой данных с помощью веб-консоли, обеспечивая надежность IT-инфраструктуры **любого** размера

Производим установку файла по ссылке:
```bash
https://disk.yandex.ru/d/5CYLCX2KTiq4eA
```
После установки, если первое действие проводилось с локального компьютера, перекидываем наш файл на сервер, зайдя в PowerShell на локальном компьютере, выполним следующую команду:
```bash
cd C:\Users\ИМЯ_ПК\Download
cp .\CyberBackup_18_64-bit.x86_64 user@IP_Сервера:/home/user
```
Когда файл будет успешно передан, переходим на сам сервер (в моем случае, он работает на ALT Linux 11):
```bash
chmod +x /home/user/CyberBackup_18_64-bit.x86_64
cd /home/user
./CyberBackup_18_64-bit.x86_64
Далее - проходите установку
```
Установка пройдет - переходим в браузер, с компьютера администратора (или любого другого, имеющего доступ к браузеру):
```bash
http://IP-Сервера:9877
```
На клиентах при установке просто убираем галочку 
```bash
Было: [ * ] Management Server
Стало: [ ] Management Server
```
На следующем шаге, просто вводим IP-адрес нашего третьего сервера и пользователя, под которым мы заходили в браузере
```bash
! Пример: 
Логин: root
Пароль: toor
```
