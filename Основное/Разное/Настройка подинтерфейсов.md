Переходим к настройке интерфейса ``../enp0s8/options``:
```bash
TYPE=eth
BOOTPROTO=none
ONBOOT=yes
VLAN=yes
```
Создаем под интерфейс:
``mkdir /etc/net/ifaces/enp0s8.{10, 20, 99}``
Содержимое в ``../enp0s8.10/options``:
```bash
TYPE=vlan
ONBOOT=yes
BOOTPROTO=static
VID=10
HOST=enp0s8
```
Для остальных - аналогично, меняется только VID
