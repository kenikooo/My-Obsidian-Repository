> [!INFO] Что такое **XTerm**?
> Классический, стандартный эмулятор терминала для графической среды XWindows System, в Unix-подобных ОС. Поддерживает изменение размера шрифтов, цветов, работу с мышью и буфером обмена.

>[!WARNING] Действия производились на ОС
>`ALT Linux`

В самом начале, через терминал, был устанавливаем следующий пакет:
```bash
apt-get update && apt-get install pve-xtermjs
```
После чего, отредактировав файл по пути `/etc/default/grub`, в самом конце добавляем следующую строку:
```bash
console=tty0 console=ttyS0,115200
```
Обновляем GRUB:
```bash
update-grub
```
Отключаем ВМ, переходим в ее конфигурацию, вкладка `Hardware` -> `Add` -> `Serial Port`, жмем `Ok`.
Запускаем, используя верхнее меню в Proxmox, находим `Console`, жмем стрелочку и выбираем `XTermJS` - буфер обмена успешно заработает.

Если XTerm все равно не работает, на ВМ прописываем следующую команду:
```bash
systemctl enable --now serial-getty@ttyS0.service
```
Перезагружаем ВМ.
