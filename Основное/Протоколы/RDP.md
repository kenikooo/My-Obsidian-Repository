`Settings -> System -> Remote Desktop -> Remote Login -> Click "Unlock"`, после чего заполняем Login details: `USERNAME` и `PASSWORD`
Через терминал:
```bash
systemctl daemon-reload
systemctl restart gnome-remote-desktop
```
После проведения настроек, вы сможете подключиться по RDP, к примеру, через Remina: жмем "+", название любое произвольное. Протокол: RDP. Заполняем: в качестве сервера используем плавающий адрес (или же любой другой, если вы работаете не с облаком), имя пользователя (под которым будет совершаться подключение, он должен существовать на устройстве) и пароль к нему.

Разрешение подключения по RDP (публичный доступ):
```bash
sudo control fusermount public
```
Данная команда разрешает всем обычным пользователям системы монтировать файловые системы через FUSE. Она меняет права доступа к утилите fusermount, делая её исполняемой для всех (устанавливает публичный доступ).
